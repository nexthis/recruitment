# Install

App is [turbo monorepo](https://github.com/vercel/turbo) required: Docker, nodeJs (~18), NPM

- `cd <repository root>/apps/app`
- then `npm install`
- `cd <repository root>/apps/backend`
- then `npm install`
- run `npm run dev`
